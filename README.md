# AlbumCollage ##
# Description #
A web application to build a collage with album cover images

# Features #
- Edit collage dimensions without losing data
- Edit image size
- Select album cover images within the application
- Drag and reorder album cover images

# Upcoming features #
- Downloading collage
- Randomizing album cover images
